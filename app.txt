git remote add <별칭> <깃허브주소>                           원격 저장소 연결결

git remote --verbise                                        연결된 원격 저장소 확인

git remote rename <old-remote-name> <new-remote-name>       원격저장소 이름 변경

git remote remove <remote-name>                             원격 연결 삭제
